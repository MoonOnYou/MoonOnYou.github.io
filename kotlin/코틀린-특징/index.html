코틀린 기초문법, 코틀린 특징

May 12, 2018

코틀린은 if 와 when (자바의 switch-case)를 명령어가 아닌 표현식(하나의 값을 가지는 식)으로 간주하기 때문에 대입문(=)의 오른쪽에 사용될 수 있다. 그러나 하나의 값만 가져야 하기 때문에 if를 사용할 때 else를 반드시 구현 해야함.

ex) a 가 1 이면 “안녕하세요”라는 문자열이 text변수에 대입된다.

val text = if (a == 1) "안녕하세요" else "안녕히가세요"
